# AI Chatbot Prompting Lab Exercises

These lab exercises help students practice **iterative prompting** when
using AI tools for learning web development.

## Lab Exercises

For each lab, students see:

-   **Basic Prompt** --- a simple question\
-   **Improved Prompt** --- clearer instructions\
-   **Advanced Prompt** --- structured instructions that produce better
    AI responses

Students should follow this workflow:

```
Write Prompt → Review AI Response → Test Code → Improve Prompt → Learn the Concept
```

### Testing Code in the Browser

When AI generates HTML, CSS, or JavaScript examples, students should
**test the code to see how it actually works**.

You can copy the code into a simple **online code editor (online IDE)**
and run it in your browser.

Beginner-friendly tools include:

-   **CodePen** --- https://codepen.io\
-   **JSFiddle** --- https://jsfiddle.net\
-   **StackBlitz** --- https://stackblitz.com\
-   **Replit** --- https://replit.com

These tools allow you to:

-   paste HTML, CSS, and JavaScript
-   run the code instantly
-   experiment with changes
-   see results in real time

Testing the code helps you **understand how the example works instead of
just reading it**.

### Basic Steps for Using an AI Chatbot

#### 1. Open the Chat Interface

Go to the chatbot website and open a new conversation.

Examples:

- ChatGPT — https://chat.openai.com  
- Claude — https://claude.ai  
- Gemini — https://gemini.google.com  
- Perplexity — https://perplexity.ai  

You will see a **text box where you can type your prompt**.


#### 2. Write Your Prompt

A **prompt** is the message you send to the AI describing what you want
it to do.

Example prompts:

```
Explain how HTML links work.

Create a simple CSS example that centers a div.

Explain this JavaScript error.
```
The AI will generate a response based on your prompt.

#### 3. Read the Response Carefully

AI responses may include:

-   explanations
-   code examples
-   suggestions
-   links or references

Always read the response carefully and make sure you understand it.

>**Remember**: AI responses may **contain mistakes or outdated information**.

#### 4. Ask Follow-Up Questions

AI chatbots work best as **conversational tools**.

You can ask follow-up questions to clarify or improve the answer.

Example:

```
Explain that again but for a beginner.

Can you provide a simpler example?

Show the same example using Flexbox.
```

Follow-up prompts often produce **better explanations**.
