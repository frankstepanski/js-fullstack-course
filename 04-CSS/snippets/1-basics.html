CSS Snippets – Basic Selectors and Properties

1. Element Selector: font-size

Code (CSS + HTML):

p.example-font-size { font-size: 20px; }
<p class="example-font-size">This paragraph text is larger.</p>

Rendered result:

This paragraph text is larger.

2. Element Selector: color

Code (CSS + HTML):

h1.example-color { color: darkred; }
<h1 class="example-color">This heading is dark red</h1>

Rendered result:

This heading is dark red

3. Element Selector: background-color

Code (CSS):

body { background-color: #f9f9f9; }

Rendered result:

The whole page has a light gray background because we set background-color on the body element.

4. Element Selector: text-align: center

Code (CSS + HTML):

h2.example-center { text-align: center; }
<h2 class="example-center">Centered heading</h2>

Rendered result:

Centered heading

Code (CSS + HTML):

img.example-border { border: 3px solid #333; }
<img class="example-border" src="https://picsum.photos/120" alt="Example image">

6. Element Selector: width and height

Code (CSS + HTML):

div.example-size { width: 200px; height: 100px; background-color: #ffeaa7; }
<div class="example-size">Fixed-size box</div>

Rendered result:

Fixed-size box

7. Element Selector: Box Model (border, padding, margin)

Code (CSS + HTML):

div.example-box-model { border: 2px solid #0984e3; padding: 10px; margin: 15px 0; background-color: #dfe6e9; }
<div class="example-box-model">Box model example</div>

Rendered result:

Box model example

8. Class Selector: .highlight

Code (CSS + HTML):

.highlight { color: #d35400; background-color: #fff3e0; }
<p>Normal text and <span class="highlight">highlighted text</span>.</p>

Rendered result:

Normal text and highlighted text.

9. Class Selector: .box

Code (CSS + HTML):

.box { border: 1px solid #555; padding: 8px; margin-bottom: 8px; background-color: #ffffff; }
<div class="box">First box</div>
<div class="box">Second box</div>

Rendered result:

First box
Second box

10. Class Selector: display: inline-block

Code (CSS + HTML):

.nav-item { display: inline-block; padding: 0.25rem 0.75rem; background-color: #0984e3; color: #fff; border-radius: 999px; }
<a href="#" class="nav-item">Home</a>
<a href="#" class="nav-item">Menu</a>
<a href="#" class="nav-item">Contact</a>

Rendered result:

Home Menu Contact

11. Descendant Selector: .note p

Code (CSS + HTML):

.note p { color: #2d3436; font-size: 0.95rem; }
<div class="note">
  <p>This paragraph is inside .note and will be styled.</p>
  <p>This one too.</p>
</div>

Rendered result:

This paragraph is inside .note and will be styled.

This one too.

12. Descendant Selector: ul.example-list li

Code (CSS + HTML):

ul.example-list li { color: #6c5ce7; }
<ul class="example-list">
  <li>Item A</li>
  <li>Item B</li>
</ul>

Rendered result:

  • Item A
  • Item B

13. Child Selector: .card-title > strong

Code (CSS + HTML):

.card-title > strong { color: #e17055; text-transform: uppercase; }
<p class="card-title">Today's special: <strong>margherita pizza</strong></p>

Rendered result:

Today's special: margherita pizza
