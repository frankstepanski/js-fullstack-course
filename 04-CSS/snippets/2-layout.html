CSS Layout Snippets

1. Layout with text-align

Code (CSS):

.ta-left-box { text-align: left; }
.ta-center-box { text-align: center; }
.ta-right-box { text-align: right; }
.ta-justify-box { text-align: justify; }

Code (HTML):

<div class="ta-demo-box ta-left-box">Left aligned text</div>
<div class="ta-demo-box ta-center-box">Centered text</div>
<div class="ta-demo-box ta-right-box">Right aligned text</div>
<div class="ta-demo-box ta-justify-box">Justified text (stretches each line).</div>

Rendered result:

Left aligned text
Centered text
Right aligned text
Justified text. The browser stretches each line so the text touches both the left and right sides.

2. Positioning (relative + absolute)

Code (CSS):

.position-card { position: relative; }
.position-badge { position: absolute; top: -10px; right: -10px; }
.position-label { position: absolute; bottom: 8px; left: 8px; }

Code (HTML):

<div class="position-card">
  <span class="position-badge">New</span>
  <p>Pizza of the Day</p>
</div>

<div class="position-card">
  <span class="position-label">Limited time</span>
  <p>Special offer details go here.</p>
</div>

Rendered result:

New

Pizza of the Day

Limited time

Special offer details go here.

4. Flexbox

Code (CSS):

.flex-row-basic { display: flex; gap: 0.5rem; }
.flex-row-basic .flex-item { flex: 1; }

Code (HTML):

<div class="flex-row-basic">
  <div class="flex-item">Box 1</div>
  <div class="flex-item">Box 2</div>
  <div class="flex-item">Box 3</div>
</div>

Rendered result:

Box 1
Box 2
Box 3

Example 2: Column layout – stacked sections

Code (CSS):

.flex-column-basic { display: flex; flex-direction: column; gap: 0.5rem; }
.flex-column-section--top { background-color: #ffeaa7; }
.flex-column-section--middle { background-color: #dfe6e9; }
.flex-column-section--bottom { background-color: #fab1a0; }

<div class="flex-column-basic">

  <section class="flex-column-section flex-column-section--middle">Main content section</section>

</div>

Hero / Intro section
Main content section
Call-to-action section

5. CSS Grid

Code (HTML):

  <div class="grid-box">Right column</div>
</div>

Rendered result:

Left column
Right column

Example 2: Three equal columns

</div>

Rendered result:

Card 3

Example 3: Simple page layout – header, sidebar, main, footer

Code (CSS):

.grid-page-layout { display: grid; grid-template-columns: 1fr 2fr; }
.grid-page-header, .grid-page-footer { grid-column: 1 / -1; }

Code (HTML):

<div class="grid-page-layout">
  <header class="grid-page-header">Header</header>
  <aside class="grid-page-sidebar">Sidebar</aside>
  <main class="grid-page-main">Main content</main>
  <footer class="grid-page-footer">Footer</footer>
</div>

Rendered result:

Header
Sidebar
Main content
Footer

6. Combining Flexbox + Grid

Example 1: Grid of cards, each card uses Flexbox

Code (CSS):

.combo-grid-cards { display: grid; grid-template-columns: repeat(3, 1fr); gap: 0.75rem; }
.combo-card { display: flex; flex-direction: column; }
.combo-card button { margin-top: auto; }

Code (HTML):

<div class="combo-grid-cards">
  <article class="combo-card">
    <h4>Cheese Pizza</h4>
    <p>Simple and classic.</p>
    <button>Order</button>
  </article>
  ... more cards ...
</div>

Rendered result:

Cheese Pizza

Simple and classic.

Pepperoni

Lots of pepperoni on top.

Veggie

Loaded with fresh vegetables.

Example 2: Flex row of sections, each section uses Grid

Code (CSS):

.combo-flex-row { display: flex; flex-wrap: wrap; gap: 0.75rem; }
.combo-flex-section { display: grid; grid-template-columns: 1fr 1fr; }
.combo-flex-section h4 { grid-column: 1 / -1; }

Code (HTML):

<div class="combo-flex-row">
  <section class="combo-flex-section">
    <h4>Toppings</h4>
    <span>Mushrooms</span>
    <span>Olives</span>
    <span>Onions</span>
    <span>Peppers</span>
  </section>
  <section class="combo-flex-section">...</section>
</div>

Rendered result:

Toppings

Mushrooms Olives Onions Peppers

Sizes

Small Medium Large XL
