HTML Snippets

1. Basic Element Structure

Code:

<p>This is a paragraph.</p>

Rendered result:

This is a paragraph.

2. Headings

Code:

<h1>This is an H1 heading</h1>
<h2>This is an H2 heading</h2>
<h3>This is an H3 heading</h3>

Rendered result:

This is an H1 heading

This is an H2 heading

This is an H3 heading

3. Nesting Elements

Code:

<p>This is a <strong>bold</strong> word inside a paragraph.</p>

Rendered result:

This is a bold word inside a paragraph.

4. Attributes

Code:

<p id="intro">Hello!</p>
<a href="https://example.com">Visit Example</a>

Rendered result:

Hello!

Visit Example

5. Several Lines of Text (List-Like)

Code:

<p>First item</p>
<p>Second item</p>
<p>Third item</p>

Rendered result:

First item

Second item

Third item

6. Images

Code:

<img src="cat.png" alt="A cute cat">

Rendered result:

(If the image file existed, it would appear below.)

7. Line Break and Horizontal Rule

Code:

<p>First line<br>Second line</p>
<hr>
<p>Text after the horizontal rule.</p>

Rendered result:

First line
Second line

Text after the horizontal rule.

8. Comments

Code:

<!-- This is a comment. It will not show on the page. -->
<p>This paragraph will be visible.</p>

Rendered result:

This paragraph will be visible.
